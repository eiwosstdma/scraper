## How to use ?
> **Warning**
>
> Works only with Node >= 18.0.0 (fetch usage)

### Install it
```powershell
  npm i mtg-scrapper
```
### Import it in your project
```
  import { LinkBuilderMtgo, ScraperParserMtgo } from 'mtg-scrapper';
  import { Deck } from 'mtg-scrapper' 
```
